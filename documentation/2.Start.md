### Docker
#### Go to the docker container
    Check Docker Status
        ```
        service docker status
        ```

    If not running/start then
        ```
        service docker start
        ```

        
    Container
        ```
        docker-compose ps
        ```

    or
        ```
        docker-compose ps -a
        ```

    *Collect The Docker Container Name*
        ```
        docker exec -it <containerName> bash
        ```

    Example:
        ```
        docker exec -it docker-nmp_php-fpm_1 bash
        ```
